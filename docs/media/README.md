# Media Assets

Add demo captures for README and portfolio usage.

Recommended files:
- `fleet-overview.png`
- `live-map-delivery.png`
- `live-map-warehouse.png`
- `robot-detail.png`
- `incidents.png`
- `replay.gif`

Guidelines:
- Prefer 16:9 captures with readable text at 100% zoom.
- Keep one consistent dataset/run during capture to avoid UI drift across images.
- Use anonymized/mock data only.
